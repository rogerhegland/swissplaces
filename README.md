# swissplaces
Deliver zip for cantons, regions and districts

## Examples
### Cantons

#### Get postalcodes of all cantons
`Canton::getPostalcodes()`

#### Get postalcodes of desired canton(s)

`Canton::getPostalcodes('TG,SG,AG')`

or

`Canton::getPostalcodes(array('TG', 'SG', 'AG'))`
